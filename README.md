# README

## Compilation du projet

``
mvn compile war:war
``

## Demarrer TOMCAT sous Ubuntu (Unix)

``
sudo /opt/tomcat/latest/bin/startup.sh
``

## Arreter TOMCAT sous Ubuntu (Unix)

``
sudo /opt/tomcat/latest/bin/shutdown.sh
``
